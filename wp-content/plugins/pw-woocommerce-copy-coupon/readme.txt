Simply adds a Copy button to WooCommerce coupons to make it easier to create new coupons!

== Description ==

PW WooCommerce Copy Coupon adds a Copy link to the default WooCommerce Coupon pages. This makes it possible to duplicate an existing coupon.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pw-woocommerce-copy-coupon` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
